# Static Web Scraper

## Description

A Python-based scraper that extracts structured product data from a paginated website using HTTP requests and HTML parsing.

---

## Features

* Multi-page scraping using URL patterns
* Structured data extraction (title, price, rating, availability)
* Data cleaning and normalization
* CSV and JSON output

---

## Tech Stack

* Python
* Requests
* BeautifulSoup
* Pandas

---

## How to Run

```bash
pip install -r requirements.txt
python main.py
```

---

## Output

* `data/books.csv`
* `data/books.json`

---

## Key Learning

* HTML structure analysis
* Pagination handling
* Data transformation
